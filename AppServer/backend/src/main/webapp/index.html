API List for SSWAP Cottage Booking

Deployment Note: Host the application in **Tomcat 9.\* with port 8080** and use **sswapdemo.war** as the WAR name for the following URLs to work.

API 1: Get SSWAP RDG for Cottage Search

Retrieves the **Resource Definition Graph (RDG)** in XML format.

URL: http://localhost:8080/sswapdemo/CottageService/api/searchCottageService

Type: GET

It should return the RDG content in **XML format**.

API 2: Get Cottage List based on Search Criteria

Sends search criteria using the **Response/Request Graph (RRG)** and retrieves the results.

URL: http://localhost:8080/sswapdemo/CottageService/api/searchCottageService

Type: POST

Headers:

Content-Type: text/turtle

Accept: text/turtle

Body:

The request body must contain the **RRG** (Request/Response Graph) in **Turtle format (raw text)**.
