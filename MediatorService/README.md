# Mediator Service

## Project Structure

This folder contains the Mediator Service for Task 7-2 of the Cottage Booking application.

### Folder: CottageMediator_Client

Complete Eclipse project for the Mediator Service Client component.

**Components:**
- Frontend: HTML + JavaScript user interface
- Backend: Java Servlet for handling requests

**Technologies:**
- Java 8
- Apache Tomcat 9.0
- Servlet API
- HTML5/CSS3/JavaScript
- AJAX

**Setup:**
1. Import the `CottageMediator_Client` folder into Eclipse as an existing project
2. Configure Java 8 JDK
3. Add Tomcat 9.0 runtime
4. Deploy and run
